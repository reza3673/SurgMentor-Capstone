# Day 4 - Security and Evaluation

## Core Idea

Security and evaluation are first-class components.

They are not optional.

## Security Principles

### Least Privilege

Grant only required permissions.

### Guardrails

Prevent unsafe outputs.

### Human in the Loop

Require oversight for high-risk actions.

### Prompt Sanitization

Validate user inputs.

### Context Hygiene

Prevent prompt injection.

## Evaluation

Evaluate:

- Accuracy
- Safety
- Tool Selection
- User Value
- Workflow Quality

## SurgMentor Safety

Educational use only.

No direct diagnosis.

No direct treatment recommendations.

No patient-identifiable information.

No fabricated case details.
