# Day 3 - Agent Skills

## Core Idea

Skills are reusable capabilities.

A skill is not a model.

A skill is specialized behavior.

## Benefits

- Smaller context
- Better scalability
- Easier maintenance
- Easier evaluation

## SurgMentor Skills

### Case Retrieval Skill

Find relevant surgical cases.

### OSCE Examiner Skill

Conduct OSCE sessions.

### Clinical Reasoning Skill

Guide diagnostic reasoning.

### Feedback Skill

Evaluate performance.

### Study Planner Skill

Generate remediation plans.

## Design Principle

Skills should be composable.

Complex workflows are built from multiple skills.
