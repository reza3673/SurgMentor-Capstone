# Day 1 - The New SDLC with Vibe Coding

## Core Idea

Software engineering is shifting from writing code to expressing intent.

Developers increasingly act as:

- Architects
- Conductors
- Orchestrators

rather than traditional coders.

## Agentic Engineering

Traditional:

Problem → Code → Product

Agentic:

Problem → Intent → Agent System → Product

## Context Engineering

The most important skill.

An agent's quality depends on:

- Context
- Instructions
- Memory
- Tools
- Evaluation

not only the model.

## Factory Model

Developers build systems that produce software.

The focus shifts from implementation to workflow design.

## Harness Engineering

The harness contains:

- Prompts
- Tools
- Memory
- Routing
- Evaluation
- Security

## Application to SurgMentor

Build an educational agent system.

Do not build a simple chatbot.
